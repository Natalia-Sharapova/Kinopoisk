# Kinopoisk

## Приложение - клон Кинопоиска. Был использован публичный API.

## Стек: 
### -Swift, 
### -UIKit, 
### -MVC,
### -Podfile SDWebImage, позволяющий кешировать изображения, 
### -Верстка кодом,
### -CoreData,
### -URLSession,
### -UIAnimation,
### -Protocols,
### -NSNotification,
### -UIContextMenuConfiguration,
### -WKWebView,
### -StretchyTableHeaderView,
